About

Ctrl-Mac Simulator as the name suggests, is a visual simulator for the Ctrl-Mac protocol, based on the paper by Bhatia et. al..

It has been developed by Edoardo Luciani as part of the dissertation for the graduation year at University of Glasgow.

The code is open source and hosted on GitHub, where contributions are welcome! There you can find instructions on how to run the simulation locally, how to access the backend api directly, and how to simulate scenarios using the command line tool.
Feel free to report any bugs on the Issues page. For personal contact my email can be found on my GitHub profile.
